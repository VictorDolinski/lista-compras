# Lista de Compras - React Native

Aluno: Victor Matheus Dolinski de Abreu

## Descrição

Aplicativo mobile desenvolvido em React Native com Expo para gerenciamento de lista de compras.

Funcionalidades:
- adicionar itens;
- listar itens;
- marcar como comprado;
- excluir itens;
- salvar dados localmente com AsyncStorage.

## Como executar

```bash
npm install
npx expo start --web
```

## Tecnologias utilizadas

- React Native
- Expo
- AsyncStorage
